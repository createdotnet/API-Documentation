Order Products
=============

*Access Levels*    
__Group:__ shop, order_managment     
__Resource:__ order_products

List all products in an order
-------------------

{% highlight php %}
GET 	https://api.create.net/orders/{order_id}/products
{% endhighlight %}

### Response

{% highlight php %}
Status: 200 OK
